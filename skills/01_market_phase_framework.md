# Skill 01：行情阶段划分框架

## 概述

阿狼使用自创的"3-X"阶段体系来描述A股大盘所处的行情阶段，并据此决定仓位和操作策略。该体系以上证指数的技术形态、量能变化和资金行为为判断依据。

---

## 阶段定义

### 3-1：底部确认阶段（磨底期）
- **特征**：指数在低位震荡，量能持续萎缩，无明显方向
- **操作**：轻仓观望，等待确认信号，不主动加仓
- **关键信号**：日线级别出现 W 底形态 + 红三兵

### 3-2：突破确认阶段（启动期）
- **特征**：指数突破关键颈线位，量能开始放大，红三兵确认
- **操作**：逐步建仓，买入主线题材
- **结束确认条件**：日线 W 底后红三兵 → 突破颈线加速 → 关键点位（如 4006）站稳三天后量能暴涨
- **原文**（2026-01-12）：
  > "3-2 的结束确认条件是什么：日线级别 W 底后的红三兵表现，突破颈线的加速，4006 站稳三天后量能的暴涨，吃技术流和踏空资金的进场溢价，这些都是一步步来的。"

### 3-3：主升浪阶段（加速期）
- **特征**：量能持续放大，主线题材加速上涨，踏空资金不断涌入
- **操作**：满仓轮动（75% 以上），主升加速浪不言顶
- **第一阶段目标区间**（本轮）：4317–4375
- **结束信号**：放量转缩量 + 收黑 K 跌破 5 日线 → 短期见顶，减仓避险
- **原文**（2026-01-12）：
  > "主升加速浪开启不言顶。接盘的量不够了就会调整，那就是放量转缩量，收黑 K 跌破 5 日线那就是短期见顶，而且只是短期，到时候减仓避一下就行。"

### 3-4：调整阶段（震荡期）
- **特征**：量能萎缩，指数在关键支撑位附近震荡，主线题材退潮
- **操作**：降至 50% 仓位，减少操作，以持仓等待为主；做低位 T
- **原文**（2026-01-21 前后）：
  > "控制仓位，减少操作，持仓等，在回暖前预期低一点。"

### 3-5：趋势延续 / 二次加速阶段
- **特征**：调整结束后再次放量突破，形成新一轮主升
- **操作**：重新加仓至 75% 以上
- **注意**：3-3 和 3-5 的资金会在关键点位发生剧烈多空博弈

---

## 阶段判断决策树

```
观察当前大盘：
1. 量能是否持续放大？
   - 是 → 进入 3-3 或 3-5 判断
   - 否（缩量）→ 进入 3-1 或 3-4 判断

2. 是否突破关键颈线/前高？
   - 是 + 放量 → 3-2 确认 / 3-3 开启
   - 否 → 3-1 磨底或 3-4 调整

3. 主线题材是否在加速？
   - 是 → 3-3 主升浪，满仓轮动
   - 否 → 3-4 调整，降仓等待

4. 是否出现放量转缩量 + 黑 K 破 5 日线？
   - 是 → 短期见顶，减仓至 50-60%
```
